## Ports
- `0x03c4` SC Index
- `0x03c5` SC Data

- `0x03ce` GC Index
- `0x03cf` GC Data

- `0x03d4` CRTC Index
- `0x03d5` CRTC Data

- `0x03c2` Misc Output

- `0x03da` Input Status 1

## Registers
- `0x00` Set/Reset Value
- `0x01` Enable Set/Reset
- `0x02` Map Mask
- `0x05` Mode
